
#  Loan Approval Prediction using Machine Learning

##  Overview

This project is a Machine Learning-based Loan Approval Prediction System developed using **Python** in **Visual Studio Code (VS Code)** with **Jupyter Notebook**. The objective is to predict whether a loan application should be approved or rejected based on applicant information.

---

##  Features

* Data Understanding
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Correlation Heatmap
* Feature Encoding & Scaling
* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier
* Model Evaluation
* Model Comparison

---

##  Dataset

The project uses the **Loan Approval Prediction Dataset** from Kaggle.

**Target Variable:**

* `loan_status`

**Input Features:**

* Loan ID
* Number of Dependents
* Education
* Self Employed
* Annual Income
* Loan Amount
* Loan Term
* CIBIL Score
* Residential Assets Value
* Commercial Assets Value
* Luxury Assets Value
* Bank Asset Value

##  Evaluation Metrics

The models were evaluated using:

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

---

##  Results

After comparing all models, **Random Forest Classifier** achieved the highest performance and was selected as the best model for loan approval prediction.
